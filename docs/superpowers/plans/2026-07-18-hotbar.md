# Hotbar & Block Selection Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement a 9-slot hotbar HUD at the bottom of the screen with selectability (keys 1-9 & scroll wheel), creative/survival mode switching (key G), placement/digging item rules, and textured thumbnail slot icons.

**Architecture:** A new module `src/inventory.rs` tracks slots, selection, and game modes. We create a textured 2D UI pipeline to render block thumbnails flat in NDC space by sampling from the existing texture atlas. Non-textured UI elements (bar background, borders) and stack count text are drawn using the existing color UI pipelines.

**Tech Stack:** Rust, wgpu, winit, WGSL

```rust
use crate::world::BlockType;

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum GameMode {
    Creative,
    Survival,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub struct ItemStack {
    pub block_type: BlockType,
    pub count: u32,
}

pub struct Hotbar {
    pub slots: [Option<ItemStack>; 9],
    pub selected: usize, // 0..8
}

impl Hotbar {
    pub fn new_creative() -> Self {
        let mut slots = [None; 9];
        let creative_blocks = [
            BlockType::Grass,
            BlockType::Dirt,
            BlockType::Stone,
            BlockType::OakLog,
            BlockType::OakPlanks,
            BlockType::Glass,
            BlockType::Cobblestone,
            BlockType::Water,
            BlockType::Torch,
        ];
        for (i, &block) in creative_blocks.iter().enumerate() {
            slots[i] = Some(ItemStack {
                block_type: block,
                count: 64,
            });
        }
        Self { slots, selected: 0 }
    }

    pub fn get_selected_block(&self) -> Option<BlockType> {
        self.slots[self.selected].map(|stack| stack.block_type)
    }

    pub fn add_item(&mut self, block: BlockType) -> bool {
        if block == BlockType::Air {
            return false;
        }
        // Try to add to existing stack
        for slot in self.slots.iter_mut() {
            if let Some(stack) = slot {
                if stack.block_type == block && stack.count < 64 {
                    stack.count += 1;
                    return true;
                }
            }
        }
        // Try to add to empty slot
        for slot in self.slots.iter_mut() {
            if slot.is_none() {
                *slot = Some(ItemStack {
                    block_type: block,
                    count: 1,
                });
                return true;
            }
        }
        false
    }

    pub fn use_selected_item(&mut self, infinite: bool) {
        if infinite {
            return;
        }
        if let Some(stack) = &mut self.slots[self.selected] {
            if stack.count > 1 {
                stack.count -= 1;
            } else {
                self.slots[self.selected] = None;
            }
        }
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_hotbar_creative_init() {
        let hotbar = Hotbar::new_creative();
        assert_eq!(hotbar.selected, 0);
        assert_eq!(hotbar.get_selected_block(), Some(BlockType::Grass));
        assert_eq!(hotbar.slots[0].unwrap().count, 64);
    }

    #[test]
    fn test_hotbar_use_item() {
        let mut hotbar = Hotbar::new_creative();
        // Creative mode shouldn't consume
        hotbar.use_selected_item(true);
        assert_eq!(hotbar.slots[0].unwrap().count, 64);

        // Survival mode should consume
        hotbar.use_selected_item(false);
        assert_eq!(hotbar.slots[0].unwrap().count, 63);

        hotbar.slots[0].as_mut().unwrap().count = 1;
        hotbar.use_selected_item(false);
        assert!(hotbar.slots[0].is_none());
        assert_eq!(hotbar.get_selected_block(), None);
    }

    #[test]
    fn test_hotbar_add_item() {
        let mut hotbar = Hotbar {
            slots: [None; 9],
            selected: 0,
        };
        assert!(hotbar.add_item(BlockType::Stone));
        assert_eq!(hotbar.slots[0].unwrap().block_type, BlockType::Stone);
        assert_eq!(hotbar.slots[0].unwrap().count, 1);

        assert!(hotbar.add_item(BlockType::Stone));
        assert_eq!(hotbar.slots[0].unwrap().count, 2);

        // fill slot 0 to 64
        hotbar.slots[0].as_mut().unwrap().count = 64;
        // next Stone should go to slot 1
        assert!(hotbar.add_item(BlockType::Stone));
        assert_eq!(hotbar.slots[1].unwrap().block_type, BlockType::Stone);
        assert_eq!(hotbar.slots[1].unwrap().count, 1);
    }
}
```

In `src/shader.wgsl`, append:
```wgsl
struct TexturedUiVertexInput {
    @location(0) position: vec3<f32>,
    @location(1) tex_coords: vec2<f32>,
    @location(2) color: vec4<f32>,
};

struct TexturedUiVertexOutput {
    @builtin(position) clip_position: vec4<f32>,
    @location(0) tex_coords: vec2<f32>,
    @location(1) color: vec4<f32>,
};

@vertex
fn vs_textured_ui(model: TexturedUiVertexInput) -> TexturedUiVertexOutput {
    var out: TexturedUiVertexOutput;
    out.clip_position = vec4<f32>(model.position, 1.0);
    out.tex_coords = model.tex_coords;
    out.color = model.color;
    return out;
}

@fragment
fn fs_textured_ui(in: TexturedUiVertexOutput) -> @location(0) vec4<f32> {
    let tex_color = textureSample(t_diffuse, s_diffuse, in.tex_coords);
    return tex_color * in.color;
}
```

  ```rust
  pub struct TexturedUiVertex {
      pub position: [f32; 3],
      pub tex_coords: [f32; 2],
      pub color: [f32; 4],

          wgpu::VertexBufferLayout {
              array_stride: std::mem::size_of::<TexturedUiVertex>() as wgpu::BufferAddress,
  ```

### Task 5: Implement Hotbar HUD Rendering

**Files:**
- Modify: `src/state.rs`

- [ ] **Step 1: Modify `State::render` to generate and draw HUD geometry**

In `src/state.rs`'s `render()` function:
- Define HUD vertices arrays inside `if !self.is_paused` branch:
  ```rust
  let mut ui_vertices = Vec::new();
  let mut ui_line_vertices = Vec::new();
  let mut ui_textured_vertices = Vec::new();
  ```
- Compute layout:
  - `let aspect = self.size.width as f32 / self.size.height as f32;`
  - Slot width $w_{ndc} = 0.08$.
  - Slot height $h_{ndc} = 0.08 \cdot \text{aspect}$.
  - Gap $g_{ndc} = 0.01$.
  - Start x coord: $x_{start\_base} = -0.40$.
- Add Background Bar:
  - From $x \in [-0.41, 0.41]$ and $y \in [-0.96, -0.94 + h_{ndc}]$.
  - Color `[0.1, 0.1, 0.1, 0.6]`. Push 6 vertices to `ui_vertices`.
- For each slot $i \in [0, 8]$:
  - Let $x_0 = x_{start\_base} + i \cdot (w_{ndc} + g_{ndc})$.
  - Let $x_1 = x_0 + w_{ndc}$.
  - Let $y_0 = -0.95$.
  - Let $y_1 = y_0 + h_{ndc}$.
  - Draw wireframe border (push 8 vertices to `ui_line_vertices` forming a line-loop rect). White `[1.0, 1.0, 1.0, 1.0]` for selected slot ($i == \text{self.hotbar.selected}$), else dark gray `[0.3, 0.3, 0.3, 0.8]`.
  - Draw block thumbnail (if slot contains item stack):
    - Retrieve texture indices: `let (col, row) = stack.block_type.get_face_tex_index(0);`
    - Calculate UV coordinates:
      - `let u0 = col as f32 * 0.0625;`
      - `let u1 = (col + 1) as f32 * 0.0625;`
      - `let v0 = row as f32 * 0.0625;`
      - `let v1 = (row + 1) as f32 * 0.0625;`
    - Scale thumbnail to fit within slot with a margin:
      - `let margin_x = 0.015;`
      - `let margin_y = 0.015 * aspect;`
      - `let tx0 = x_0 + margin_x;`
      - `let tx1 = x_1 - margin_x;`
      - `let ty0 = y_0 + margin_y;`
      - `let ty1 = y_1 - margin_y;`
    - Push 6 vertices to `ui_textured_vertices` using `TexturedUiVertex` with color `[1.0, 1.0, 1.0, 1.0]`.
  - Draw count text (if stack count > 1):
    - Use `add_string_lines` with smaller dimensions (`char_w = 0.008`, `char_h = 0.016`, `spacing = 0.003`) centered around $x = x_1 - 0.035$, $y = y_0 + 0.015 \cdot \text{aspect}$. Pass a reference to `ui_line_vertices`.
- Add a game mode status indicator on the top or above the hotbar:
  - Print the current game mode (e.g. `CREATIVE` or `SURVIVAL`) centered above the hotbar (e.g. at $y = -0.75$).
- Write buffers:
  - `let ui_textured_vert_len = ui_textured_vertices.len().min(1024);`
  - Write `ui_textured_vertices[..ui_textured_vert_len]` to `self.ui_textured_vertex_buffer`.
  - Write `ui_vertices` to `self.ui_vertex_buffer` (merging with pause menu buffers).
  - Write `ui_line_vertices` to `self.ui_line_vertex_buffer`.
- Inside the render pass encoder (in `State::render`):
  - In `if !self.is_paused` branch:
    - First set pipeline `self.ui_textured_pipeline`.
    - Set bind group 0 to `self.camera_bind_group`.
    - Set vertex buffer 0 to `self.ui_textured_vertex_buffer`.
    - Draw: `render_pass.draw(0..self.num_ui_textured_vertices, 0..1);`
    - Then set pipeline `self.ui_pipeline`.
    - Set vertex buffer 0 to `self.ui_vertex_buffer`.
    - Draw: `render_pass.draw(0..self.num_ui_vertices, 0..1);`
    - Then set pipeline `self.ui_line_pipeline`.
    - Set vertex buffer 0 to `self.ui_line_vertex_buffer`.
    - Draw: `render_pass.draw(0..self.num_ui_line_vertices, 0..1);`
