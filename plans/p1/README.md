## 建議執行順序

1. **任務 6** (背包/合成) — 生存循環的核心
2. **任務 7** (工具) — 依賴背包系統
3. **任務 11** (洞穴/礦脈) — 豐富地下探索
4. **任務 8** (生命/飢餓) — 生存壓力
5. **任務 9** (日夜循環) — 敵對生物依賴光照等級
6. **任務 10** (敵對生物) — 需要日夜 + 生命系統
7. **任務 12** (音效) — 最後潤色

## 整體驗證標準

完成 P1 全部任務後，遊戲應該：
1. ✅ 有完整的**背包和合成系統**，能合成工具
2. ✅ **不同工具**挖掘不同方塊有速度差異
3. ✅ 有**生命值和飢餓值**，能死能活
4. ✅ 有**日夜交替**，夜晚會變暗
5. ✅ 夜晚出現**敵對生物**，能戰鬥
6. ✅ 地下有**洞穴和礦物**，有探索的動力
7. ✅ 有**音效**，不再是無聲遊戲
8. ✅ **完整的生存循環**: 挖礦→合成→生存→探索
