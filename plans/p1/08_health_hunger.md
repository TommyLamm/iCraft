## 8.1 玩家生命值

### 新增/修改文件
- **[NEW]** `src/player.rs` — 玩家狀態（生命/飢餓/經驗）
- **[MODIFY]** `src/physics.rs` — 摔落傷害計算
- **[MODIFY]** `src/state.rs` — HUD 渲染 + 死亡畫面

### 實現細節
```rust
pub struct PlayerState {
    pub health: f32,          // 0~20 (10 顆心)
    pub max_health: f32,
    pub hunger: f32,          // 0~20 (10 個雞腿)
    pub saturation: f32,      // 飽食度
    pub armor: f32,           // 0~20
    pub oxygen: f32,          // 0~300 ticks (水下氧氣)
    pub is_dead: bool,
    pub invulnerable_ticks: i32, // 受傷後無敵幀
}
```

## 8.2 摔落傷害
- [ ] 記錄玩家離開地面時的 Y 座標
- [ ] 著地時計算摔落距離：`fall_distance = start_y - land_y`
- [ ] 超過 3 格造成傷害：`damage = fall_distance - 3`
- [ ] 掉入虛空 (Y < -64) 持續受傷

---

## 8.3 飢餓系統
- [ ] 移動/挖掘/跳躍消耗飢餓值
- [ ] 衝刺消耗更多飢餓值
- [ ] 飢餓值 > 18 時自然回血
- [ ] 飢餓值 = 0 時持續掉血（簡單難度到 10HP，普通到 1HP，困難到死）
- [ ] 食物物品：右鍵食用恢復飢餓值和飽食度

---

## 8.4 死亡與重生
- [ ] 生命值歸零時顯示死亡畫面
- [ ] 死亡畫面：「你死了！」文字 + 「重生」按鈕 + 死因
- [ ] 重生：回到世界出生點，重置生命值和飢餓值
- [ ] 死亡時掉落所有背包物品
