## 14.2 各生物實現

### 豬 (Pig)
- [ ] 粉色長方體模型 (身體+頭+四腿)
- [ ] 掉落: 生豬肉 (烤豬排 if 火焰致死)
- [ ] 用胡蘿蔔餵養/繁殖

### 牛 (Cow)
- [ ] 黑白色模型 (身體+頭+角+四腿)
- [ ] 掉落: 生牛肉, 皮革
- [ ] 用小麥餵養/繁殖
- [ ] 用桶右鍵可擠奶

### 羊 (Sheep)
- [ ] 白色模型 (帶羊毛層)
- [ ] 掉落: 羊毛, 生羊肉
- [ ] 用剪刀可剪羊毛 (不殺死)
- [ ] 吃草回覆羊毛
- [ ] 可用染料染色

### 雞 (Chicken)
- [ ] 小型白色模型
- [ ] 掉落: 生雞肉, 羽毛
- [ ] 每 5~10 分鐘產一顆蛋
- [ ] 從高處掉落緩慢下降 (拍翅膀)
