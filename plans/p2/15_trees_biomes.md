## 15.2 生態群系系統
```rust
pub enum Biome {
    Plains,      // 平原: 草地, 偶爾橡木
    Forest,      // 森林: 密集橡木/白樺
    Desert,      // 沙漠: 沙子, 仙人掌, 無雨
    Taiga,       // 針葉林: 雲杉, 雪
    Swamp,       // 沼澤: 平坦, 水多, 橡木
    Mountains,   // 山地: 高海拔, 石頭裸露
    Ocean,       // 海洋: 大片水域
}
```

### 子任務清單
- [ ] 使用 2D Voronoi / 大尺度 Perlin 噪聲分配生態群系
- [ ] 每種生態群系決定:
  - 地表方塊 (草地/沙子/雪/石頭)
  - 地形高度曲線 (平坦/起伏/山地)
  - 植被類型和密度
  - 生物生成類型
- [ ] 生態群系過渡: 邊界平滑插值避免突變
