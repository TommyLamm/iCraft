## 建議執行順序

1. **任務 1** (多 Chunk) — 所有後續內容的基礎架構
2. **任務 2** (方塊/紋理) — 視覺辨識度
3. **任務 3** (光照) — 核心視覺體驗
4. **任務 4** (天空/霧效) — 快速視覺提升
5. **任務 5** (快捷欄) — 基礎交互

## 整體驗證標準

完成 P0 全部任務後，遊戲應該：
1. ✅ 有一個**無限延伸**的方塊世界（至少 render distance = 8）
2. ✅ 有 **15+ 種方塊**，每種都有辨識度高的紋理
3. ✅ 有**光照系統**，洞穴變暗、火把能照亮
4. ✅ 有**天空漸變和霧效**，遠景自然
5. ✅ 能通過**快捷欄選擇不同方塊**來放置
